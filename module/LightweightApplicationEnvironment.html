Lightweight Application Environment


Overview


Lightweight Application Environment (LAE) is a Cisco Cloud Computing Platform as a Service (PaaS). LAE gives developers a platform in the cloud to build, test, deploy, and run applications. LAE provides the infrastructure, middleware, and management... setting you free to focus on what you do best: designing and coding applications. LAE leverages the open source ecosystem to provide key platform services and enterprise-class middleware platforms.

Programming languages/frameworks : Java (Tomcat/JBoss), Ruby, PHP, Perl, Python, Node.js.
Databases: MySQL, PostgreSQL, MongoDB
Integrated developer tools through SDaaS
